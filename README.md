This is an AI assistant that can open separate assistants automatically so you can get to working as fast as possible

Note-This is incredibly alpha,You will need to edit the script to add new projects,will add an UI and make it an exe so it starts on startup.This is just a small teaser kinda.(There are tons of bugs and problems,if youre running into some i would gladly take in the feedback)

Instructions:
Download all the extra libraries
Run using python 3.12.2
Commands: most of the commands will be given by the cli
for changing the location or text thing just enter "change text or loc"
Then after running, CTRL+C will exit the program(The program may take a few seconds to close,during that time you might notice your computer getting laggy.Not to worry its because of the closing code.)
